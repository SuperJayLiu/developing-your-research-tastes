# Turn Data Limits Into Design

## One-Sentence Skill

Use this skill to turn Data Limits Into Design when doing so will change a concrete research decision rather than merely rename the topic.

## Trigger

Use the skill when a project depends on an observed variable that may not yet measure the intended concept. The trigger matters because research taste is relative: the same move is useful in one project and distracting in another.

## Research Move

The move is to turn Data Limits Into Design by writing the project in terms of a question, a mechanism or design, an evidence standard, and a boundary. A mature use of the skill should make one of those four pieces clearer.

## Evidence Anchor

Anchor the skill in papers, scholar pages, journal articles, or uploaded PDFs before treating it as finished. The anchor should show where the move appears in question choice, mechanism, evidence, or framing.

## Good Taste Standard

Good taste means the skill changes the project. The before-and-after note should show a sharper question, cleaner comparison, better measure, stronger mechanism, narrower contribution claim, or more persuasive introduction.

## Bad Taste Boundary

This becomes bad taste when the measure is treated as valid because it is available, not because it has been justified. The boundary should be kept with the skill because it tells the researcher when imitation has stopped helping.

## Economics Applications

In economics, this skill is useful when it clarifies a mechanism, design, policy comparison, welfare claim, or interpretation of heterogeneous effects.

## Finance Applications

In finance, this skill is useful when it separates risk, beliefs, frictions, constraints, incentives, and market plumbing rather than treating price patterns as self-explanatory.

## Scholar Examples

Use the scholar pages in Chapter 04 as the evidence source. A good example should show the move across more than one paper or research program.

## Practice Exercise

Apply the skill to a live project in one page. Write the trigger, the move, the closest alternative explanation, the evidence that would change a skeptical reader's mind, and the narrowest honest contribution claim.

## Feedback Signal

The skill worked if the project now has a better decision rule. It is still immature if the exercise only produces topic words or a sentence of admiration.
