# Idea Generation Drills

## Drill

Choose one active research idea and apply the exercise below.

## Steps

1. Write the current version of the idea.
2. Identify the weakest part of the idea.
3. Apply one skill from `skills/`.
4. Write the revised version.
5. State what evidence would make you update.

## Output

```text
Before:
After:
Skill used:
Evidence needed:
Taste update:
```
