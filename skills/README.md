# Skills

Skills are the central unit of the repo.

A skill is a reusable research move.

Examples:

- Turn an anomaly into a research question.
- Find identification in institutions.
- Build a minimal model of the core friction.
- Separate risk from mispricing.
- Measure an unobservable concept.
- Design a falsification test.

Each skill should link to scholar pages, papers, journal tastes, and research steps.
