# Make research infrastructure reusable

## Source Scholar

Kenneth French

## Generalisable Skill

Turn data definitions, factor construction, and portfolio returns into shared infrastructure for the
field.

## Evidence Anchors

- Kenneth French Data Library
- Common Risk Factors
- Five-Factor Asset Pricing Model

## Taste Extraction Notes

### What the papers tend to test

The tests rely on transparent portfolio construction and repeated use across samples, markets, and
questions.

### What the abstract/introduction tends to do

The introduction often highlights that the model or data organize many existing findings.

### Why this skill travels

Use this skill by making your constructed data or code useful beyond one paper.

### Critical boundary

The boundary is that infrastructure choices embed assumptions. Users need documentation and
alternatives.

## Practice Exercise

Document one constructed variable so another researcher could use it without asking you.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
