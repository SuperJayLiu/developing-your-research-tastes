# Make selection the research object

## Source Scholar

James Heckman

## Generalisable Skill

Ask who enters the treatment, sample, program, or market, and what that selection reveals.

## Evidence Anchors

- Sample Selection Bias as a Specification Error
- Selection model papers
- Program evaluation work

## Taste Extraction Notes

### What the papers tend to test

The tests model the selection process and show how ignoring it biases interpretation.

### What the abstract/introduction tends to do

The introduction frames missing or selected observations as a substantive problem, not a technical
footnote.

### Why this skill travels

Use this skill whenever participation, observability, or treatment uptake is non-random.

### Critical boundary

The boundary is model dependence. Selection corrections require strong assumptions and exclusion
logic.

## Practice Exercise

Write the selection equation behind one dataset you use.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
