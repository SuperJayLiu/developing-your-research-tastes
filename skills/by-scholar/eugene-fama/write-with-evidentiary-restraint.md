# Write with evidentiary restraint

## Source Scholar

Eugene Fama

## Generalisable Skill

Use precise language about what return evidence can and cannot prove.

## Evidence Anchors

- Efficient Capital Markets
- Long-term returns and behavioral finance
- Factor papers

## Taste Extraction Notes

### What the papers tend to test

The empirical tests are interpreted cautiously because alternative risk, model, and sampling
explanations remain possible.

### What the abstract/introduction tends to do

The introduction often avoids dramatic causal claims and instead defines the empirical test clearly.

### Why this skill travels

Apply this when evidence is strong but interpretation is contested.

### Critical boundary

The risk is excessive conservatism that blocks new mechanisms. Restraint should not become refusal
to theorize.

## Practice Exercise

Rewrite an anomaly claim using three possible interpretations: risk, mispricing, and measurement.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
