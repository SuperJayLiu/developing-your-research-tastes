# Build benchmark models that organize debate

## Source Scholar

Eugene Fama

## Generalisable Skill

Create simple empirical models that become reference points for evaluating new anomalies and
explanations.

## Evidence Anchors

- Common Risk Factors
- Fama-French three-factor and five-factor papers
- Cross-section papers

## Taste Extraction Notes

### What the papers tend to test

The tests propose factors that explain broad return variation and then evaluate remaining anomalies.

### What the abstract/introduction tends to do

The writing is often spare and cumulative: here is the empirical regularity, here is the benchmark,
here is what it explains.

### Why this skill travels

Use this skill in any field by building a baseline model others can use to judge new claims.

### Critical boundary

The boundary is benchmark lock-in: useful models can become rituals detached from theory.

## Practice Exercise

Write the minimal benchmark model your field should beat.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
