# Write finance results with survival tests

## Source Scholar

Campbell Harvey

## Generalisable Skill

Frame a result as credible only after it survives economic motivation, out-of-sample evidence,
robustness, and implementability.

## Evidence Anchors

- Backtesting
- Expected returns studies
- Factor evaluation work

## Taste Extraction Notes

### What the papers tend to test

The empirical tests move from discovery to survival: costs, turnover, subsamples, markets, and
theory.

### What the abstract/introduction tends to do

The introduction can promise not just a new predictor but a disciplined evaluation of whether it
deserves belief.

### Why this skill travels

Apply this by making your results section a sequence of survival tests.

### Critical boundary

The risk is endless robustness without learning. Each test should map to a specific threat.

## Practice Exercise

Build a survival-test table for a new empirical finding.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
