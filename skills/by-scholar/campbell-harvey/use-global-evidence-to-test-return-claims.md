# Use global evidence to test return claims

## Source Scholar

Campbell Harvey

## Generalisable Skill

Check whether asset-pricing patterns travel across countries, markets, and regimes.

## Evidence Anchors

- Expected Returns and Volatility in 135 Countries
- Emerging market return studies
- Cross-country asset pricing work

## Taste Extraction Notes

### What the papers tend to test

The tests compare expected returns, volatility, and risk relations across many markets.

### What the abstract/introduction tends to do

The introduction frames global evidence as a way to test whether a pattern is structural or sample-
specific.

### Why this skill travels

Apply this by asking whether your finance result should hold outside the U.S. and why.

### Critical boundary

The risk is cross-market comparability and data quality. International breadth requires measurement
caution.

## Practice Exercise

Choose one anomaly and state whether it should exist globally or only in one institutional setting.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
