# Treat multiple testing as a first-order threat

## Source Scholar

Campbell Harvey

## Generalisable Skill

In fields with many tested predictors, treat false discovery as part of the research design problem.

## Evidence Anchors

- ... and the Cross-Section of Expected Returns
- Backtesting
- Factor zoo papers

## Taste Extraction Notes

### What the papers tend to test

The tests examine whether reported factors survive higher significance thresholds, out-of-sample
tests, or economic filters.

### What the abstract/introduction tends to do

The introduction frames the problem as field-level credibility, not a minor robustness issue.

### Why this skill travels

Use this skill whenever many hypotheses, signals, or models are tried.

### Critical boundary

The boundary is that stricter thresholds can reject real but weak effects. Good taste balances Type
I and Type II errors.

## Practice Exercise

List every specification choice in your project that could create hidden multiple testing.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
