# Build econometrics from data features

## Source Scholar

Robert Engle

## Generalisable Skill

Let recurrent empirical patterns in data motivate the model class.

## Evidence Anchors

- ARCH paper
- Dynamic Conditional Correlation
- Financial volatility papers

## Taste Extraction Notes

### What the papers tend to test

The tests respond to observed clustering, time-varying correlations, and persistence in financial
series.

### What the abstract/introduction tends to do

The writing starts from a visible data problem and then introduces the method as a solution.

### Why this skill travels

Apply this by listing the empirical regularities your method is designed to capture.

### Critical boundary

The risk is method proliferation without economic interpretation.

## Practice Exercise

Write the empirical feature that justifies one model choice in your project.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
