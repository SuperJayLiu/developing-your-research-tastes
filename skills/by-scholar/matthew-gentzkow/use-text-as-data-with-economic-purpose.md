# Use text as data with economic purpose

## Source Scholar

Matthew Gentzkow

## Generalisable Skill

Use NLP and text methods only after defining the economic object and behavioral link the text is
supposed to capture.

## Evidence Anchors

- Text as Data
- Media slant papers
- Political communication studies

## Taste Extraction Notes

### What the papers tend to test

The tests validate text measures against human coding, known events, behavioral outcomes, or market
incentives.

### What the abstract/introduction tends to do

The introduction makes the text method serve a substantive puzzle, not the other way around.

### Why this skill travels

Use this by writing a 'what does the text measure economically?' paragraph before modeling text.

### Critical boundary

The danger is topic-model tourism: impressive text analysis without an economic mechanism.

## Practice Exercise

For a text dataset, state the economic mechanism in one sentence and the validation test in another.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
