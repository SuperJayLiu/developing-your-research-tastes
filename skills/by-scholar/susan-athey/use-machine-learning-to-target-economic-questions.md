# Use machine learning to target economic questions

## Source Scholar

Susan Athey

## Generalisable Skill

Use prediction and flexible models to improve heterogeneity analysis, policy learning, or
measurement while keeping the economic estimand explicit.

## Evidence Anchors

- Machine Learning Methods That Economists Should Know About
- Recursive Partitioning for Heterogeneous Causal Effects
- Policy Learning papers

## Taste Extraction Notes

### What the papers tend to test

The tests combine predictive tools with causal objects such as treatment effects, counterfactual
policies, or heterogeneous responses.

### What the abstract/introduction tends to do

The introduction usually explains why off-the-shelf prediction is insufficient without causal
structure.

### Why this skill travels

Use this skill when ML can help discover heterogeneity or improve decisions, but define the economic
object first.

### Critical boundary

The danger is treating predictive accuracy as causal insight. Good taste keeps a wall between
prediction tasks and causal claims.

## Practice Exercise

Take one ML idea and state the causal estimand it helps estimate or the policy decision it helps
choose.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
