# Combine method innovation with applied payoff

## Source Scholar

Susan Athey

## Generalisable Skill

Develop methods in response to applied problems where existing tools cannot answer the decision
question well.

## Evidence Anchors

- Causal forests / heterogeneous effects work
- Policy learning work
- Economics and ML surveys

## Taste Extraction Notes

### What the papers tend to test

The tests evaluate whether the method improves estimation or decisions in settings economists care
about.

### What the abstract/introduction tends to do

The writing pattern starts with an applied challenge and then justifies the method as the needed
tool.

### Why this skill travels

Use this skill when writing technical work: state the applied decision failure that motivates the
method.

### Critical boundary

The risk is method-first writing where the application is decorative. Good taste demands a real
applied constraint.

## Practice Exercise

Write the first paragraph of a methods paper without mentioning the method name.

## How to Use This Skill in Your Own Research

1. Restate your research idea using this skill.
2. Name the theory or mechanism it implies.
3. State one empirical or theoretical test that could reject it.
4. Identify one rival explanation.
5. Write the introduction paragraph that would make the reader care.
