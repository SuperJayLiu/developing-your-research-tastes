# 04 — Research Tastes of Top Scholars

This section uses one page per scholar.

The goal is not biography. The goal is skill extraction.

For each scholar, ask:

```text
What does this researcher repeatedly do well that another researcher can learn to do?
```

## Status Labels

- `ready-to-draft`: enough open-access material appears available for a responsible first draft.
- `partial`: some material is available, but important papers may need upload.
- `unfinished-upload-needed`: do not expand heavily until canonical papers are uploaded.
