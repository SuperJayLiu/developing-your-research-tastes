# Theory Testing Lab

A theory is not ready until it faces a test that could make it fail.

## Test 1: Prediction test

Does the main prediction appear in the data?

## Test 2: Mechanism test

Does the proposed mechanism move when it should?

## Test 3: Alternative mechanism test

Can another mechanism explain the same result?

## Test 4: Boundary condition test

Where should the theory be strongest? Where should it be weakest?

## Test 5: Falsification test

Where should there be no effect?

## Test 6: Economic magnitude test

Is the effect large enough to matter?

## Test 7: Interpretation test

What exactly can be claimed, and what cannot be claimed?

Good taste is often knowing where to stop claiming.
