# Submission Fit Scorecard

Use this before deciding where a paper belongs.

| Dimension | Question | Score |
|---|---|---|
| Audience | Which journal audience cares most? | 1–5 |
| Question | Is the question central for that audience? | 1–5 |
| Contribution | What belief changes after the paper? | 1–5 |
| Method/design | Is the execution strong enough? | 1–5 |
| Mechanism | Is the economic channel clear? | 1–5 |
| Magnitude | Is the effect economically meaningful? | 1–5 |
| Positioning | Is the closest literature handled honestly? | 1–5 |
| Writing | Can the contribution be explained in one paragraph? | 1–5 |

## Decision Rule

If the score is low because the audience is wrong, change the target journal.

If the score is low because the contribution is unclear, revise the paper.
