# Journal Comparison Map

Use this page to compare journal taste environments.

| Journal type | Often rewards | Common weakness to avoid |
|---|---|---|
| General-interest economics | broad importance, credible design, clear contribution | narrow setting without general insight |
| Econometric/theory journals | conceptual precision, new method/model, formal clarity | application without methodological insight |
| Finance journals | prices, firms, investors, intermediaries, frictions, economic magnitude | result that is statistically significant but not financially meaningful |
| Quantitative finance journals | careful measurement, implementation, robustness | complexity without interpretation |

## How to Use

Before targeting a journal, write the paper's contribution in the language of that journal's audience.
